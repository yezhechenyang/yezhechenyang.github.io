【Spring Boot】Admin可视化监控
Spring Boot Actuator Admin
Spring Boot
发布日期:   2020-07-21
作者:   叶遮沉阳
文章字数:   1k
阅读时长:   5 分
阅读次数:  

1. 前言

Actuator监控器只是提供最简单的接口调用，未提供可视化。

本文介绍Admin可视化监控。

主要实现Admin服务端监控Admin客户端的运行状况。

版本说明：

<spring.boot.version>2.1.0.RELEASE</spring.boot.version>
<spring-boot-admin.version>2.1.0</spring-boot-admin.version>

2. 服务端

2.1 服务端Maven配置

注意：服务端移除移除Tomcat，添加Jetty，是为了解决Calling [asyncError()] is not valid错误，详情可见下面的标题类目5.1。

<dependency>
  <groupId>de.codecentric</groupId>
  <artifactId>spring-boot-admin-starter-server</artifactId>
</dependency>

<dependency>
  <groupId>org.springframework.boot</groupId>
  <artifactId>spring-boot-starter-web</artifactId>
  <exclusions>
    <exclusion>
      <groupId>org.springframework.boot</groupId>
      <artifactId>spring-boot-starter-tomcat</artifactId>
    </exclusion>
  </exclusions>
</dependency>

<dependency>
  <groupId>org.springframework.boot</groupId>
  <artifactId>spring-boot-starter-jetty</artifactId>
</dependency>

2.2 服务端yml配置

server:
  port: 8000

2.3 服务端启动器

使用@EnableAdminServer注解开启AdminServer服务。

@EnableAdminServer
@SpringBootApplication
public class SpringBootDemoAdminServerApplication {

    public static void main(String[] args) {
        SpringApplication.run(SpringBootDemoAdminServerApplication.class, args);
    }
}

3. 客户端

3.1 客户端Maven配置

<dependencies>
<dependency>
  <groupId>org.springframework.boot</groupId>
  <artifactId>spring-boot-starter-web</artifactId>
</dependency>

<dependency>
  <groupId>de.codecentric</groupId>
  <artifactId>spring-boot-admin-starter-client</artifactId>
</dependency>

<dependency>
  <groupId>org.springframework.boot</groupId>
  <artifactId>spring-boot-starter-security</artifactId>
</dependency>

3.2 客户端yml配置

server:
  port: 8080
  servlet:
    context-path: /demo
spring:
  application:
    # Spring Boot Admin展示的客户端项目名，不设置，会使用自动生成的随机id
    name: spring-boot-demo-admin-client
  boot:
    admin:
      client:
        # Spring Boot Admin 服务端地址
        url: "http://localhost:8000/"
        instance:
          metadata:
            # 客户端端点信息的安全认证信息
            user.name: ${spring.security.user.name}
            user.password: ${spring.security.user.password}
  security:
    user:
      name: xkcoding
      password: 123456
management:
  endpoint:
    health:
      # 端点健康情况，默认值"never"，设置为"always"可以显示硬盘使用情况和线程情况
      show-details: always
  endpoints:
    web:
      exposure:
        # 设置端点暴露的哪些内容，默认["health","info"]，设置"*"代表暴露所有可访问的端点
        include: "*"

3.3 客户端启动器

@SpringBootApplication
public class SpringBootDemoAdminClientApplication {

    public static void main(String[] args) {
        SpringApplication.run(SpringBootDemoAdminClientApplication.class, args);
    }
}

4. 测试

启动服务端与客户端。

浏览器访问服务端：http://localhost:8000/
可以直接看到启动的客户端实例列表。

然后点击菜单栏的Wallbord，便可看到图形化展示客户端实例。

点击任意一个实例，便可进入该实例的可视化监控界面。

5. 问题汇总

<dependency>
  <groupId>org.springframework.boot</groupId>
  <artifactId>spring-boot-starter-web</artifactId>
  <exclusions>
    <exclusion>
      <groupId>org.springframework.boot</groupId>
      <artifactId>spring-boot-starter-tomcat</artifactId>
    </exclusion>
  </exclusions>
</dependency>

<dependency>
  <groupId>org.springframework.boot</groupId>
  <artifactId>spring-boot-starter-jetty</artifactId>
</dependency>
